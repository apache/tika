= Apache Tika Roadmap

This page outlines the planned features and improvements for Apache Tika releases.

NOTE: All dates are in Open Source Standard Time which does not always neatly align
with traditional calendars.

== Metadata Changes in 4.x

Tika 4.x implements namespaced metadata keys to prevent overwrites and improve namespace clarity.

See link:migration/metadata-changes-4x.html[Metadata Changes in 4.x] for complete details and
migration examples.
