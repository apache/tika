= Configuration

This section covers configuring Apache Tika.

== Overview

Tika 4.x uses JSON configuration files. Configuration controls parsers, detectors,
content handlers, and other components.

NOTE: Tika 3.x and earlier used XML configuration (`tika-config.xml`). See the
xref:../migration-to-4x/index.adoc[Migration Guide] for details on converting to JSON.
