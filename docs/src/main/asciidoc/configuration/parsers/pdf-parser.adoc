== Configuration Options

// TODO: Document all PDFParser configuration options
// Reference: org.apache.tika.parser.pdf.PDFParser and PDFParserConfig

[cols="2,1,1,4"]
|===
|Option |Type |Default |Description

|`extractInlineImages`
|boolean
|false
|Extract inline images from the PDF

|`extractUniqueInlineImagesOnly`
|boolean
|true
|Only extract unique inline images (avoids duplicates)

|`sortByPosition`
|boolean
|false
|Sort text by position on page rather than stream order

|`maxMainMemoryBytes`
|long
|524288000
|Maximum bytes to use for main memory before spooling to disk

|===
